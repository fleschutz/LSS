# LSS

Execute the following steps:

(1) git clone https://github.com/fleschutz/LSS.git

(2) cd LSS

(3) gcc -O3 main.c   (when using the GCC compiler) or: clang -O3 main.c (when using the CLANG compiler)

(4) ./a.out
